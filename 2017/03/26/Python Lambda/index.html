title:Python Lambda
date:2016-08-07 10:21:16

  • lambda演算是什么？

λ演算（英语：lambda calculus，λ-calculus）是一套用于研究函数定义、函数应用和递归的形式系统。
Lambda演算包括一条变换规则（变量）和一条函数定义方程式（函数定义）。

  • Python中的lambda

Python中的lambda和lambda演算的定义类似，其语法为：

lambda 变量表达式 : 定义

e.x.

>>>f = lambda x : x+10

>>>f(10)

20

比如求sigmod函数：

>>> sigmod = lambda x : (1/(1 + math.exp(-1 * x)))
>>> sigmod(123)
1.0
>>> sigmod(-123)
3.817497188671175e-54
>>> sigmod(-125)
5.166420632837861e-55
>>> sigmod(-1)
0.2689414213699951
>>> sigmod(-32)
1.2664165549094016e-14
>>>

lambda的最大优点，就是可以不用定义一个可能『污染上下文』的函数（比如在有math包导入的情况下，声明一个自己的sin()函数）。

  • lambda与map/filter/reduce的应用

lambda可以方便的配合map/filter/reduce来处理列表数据，其中map(fun,list)是返回经过传入函数的每一个计算结果（列表），reduce(fun,list)是逐步用fun函数来归约列表的每一个元素，而filter顾名思义是一个过滤器，过滤掉所有fun(x)为False的列表元素并返回。

list(map(x,range(20)))
