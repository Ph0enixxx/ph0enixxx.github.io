title:Python实现提取图片特征
date:2016-01-27 12:58:57
思路：打开图片后将其转化为纯黑白通道，然后将像素点转化为一串二进制数字

from PIL import Image

img = Image.open("./a.jpg")
print(img.size)
#img
#
res = img.convert(mode = "1").resize((8,8))#.save("./b.bmp","BMP")
print(res)
#http://zhan.renren.com/linuxfunc?gid=3602888498023717986&checked=true
fingerPrint = ""
for i in range(1,8):
    for j in range(1,8):
        if res.load()[j,i]==0:
            fingerPrint += "0"
        else:
             fingerPrint += "1"

print(fingerPrint)

效果：


1000000000010010001000100101010110010110000011001
