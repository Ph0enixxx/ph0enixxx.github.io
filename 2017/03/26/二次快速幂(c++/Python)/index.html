title:二次快速幂(c++\Python)
date:2016-01-23 03:36:52
快速幂的思想是将指数分解，然后将几个小的指数得到的结果合并 来达到加速开方的目的。

百度百科上的二次快速幂：

#include <iostream>
int pow2(int ,int );
int main(void)
{
    std::cout<<pow2(2,7);
}
int pow2(int num,int s)
{
    int r = 1,base = num;
    while(s != 0)
    {
        if(s%2)//如果是奇数，就先把结果乘以自身（整理为一堆偶数的快速幂）
            r*=base;
        base*=base;//base变为base的平方
        s/=2;
    }
    return r;
}

计算步骤：
2的7次幂：
1.7是奇数 r = 2
2.base = 2^2 = 4
3.s = 3
4.3是奇数 r = 8
5.base = 4^4 = 16
6.s = 1
7.1是奇数 r = 128
8.s = 0 返回r结束
计算步骤只有 2， 5 ，7 三步
相比普通的7步快了接近一倍

Python:

def pow(num , s):
    r = 1
    base = num
    while s != 0 :
        if s%2 == 1:
            r *= base
        base *= base
        s /= 2
        s = int(s)
    return r
print(pow(2,9))
