title:一次奇怪的SQL错误
date:2016-03-04 02:34:15
昨晚在写插入语句的时候，发现怎么插入也不行，在sql终端执行时
出现了You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near ‘true)values (0,’1’,1)’ at line 1的错误，

而我的sql语句是这么写的：
insert into quizContest (problemId,user,true)values (0,’1’,1);

后来在舍友的提示下，修改了true字段名，语句通过了。SQL的保留字在创建表时不会提示，但是在查询和插入时可能出错。
