title:学校内部免费上网教程
date:2016-03-01 04:33:19

1.适用范围：学校内部能够需要学生账号登陆就能上网的地方（包括但不限于计算机中心、钟楼机房、部分Wifi）
2.价格 40元/永久（不包括寒暑假）
3.操作步骤：

1>下载shadowsocks软件：

地址1：http://202.194.119.110/shadow.exe

地址2：https://github.com/shadowsocks/shadowsocks/archive/2.8.2.zip

2>打开软件，双击任务栏右下角纸飞机图标

3>点击添加按钮：


4>输入购买得到的ip地址、端口、密码后点击确定

5>右键点击任务栏的纸飞机图标，选择启用系统代理，并且将代理模式设置为『全局模式』，服务器选择购买时的ip地址

6>不出意外已经可以上网了，如果不行请联系QQ：2898629332
