title:leetcode：Climbing Stairs
date:2016-05-29 14:02:59
题目很简单，抽象一下就是一个迭代法求fabnacci数列的问题

class Solution {
public:
    int climbStairs(int n) {
        int fab[10000];
        fab[0] = 1;
        fab[1] = 1;
        fab[2] = 2;
        for(int i=3;i<=n;i++)
            fab[i] = fab[i-1] + fab[i-2];
        return fab[n];
    }
};

title:leetcode：Remove Duplicates from Sorted Array
date:2016-05-31 09:47:49
class Solution {
public:
int removeDuplicates(vector& nums) {
set tmp;
for(auto x:nums)
tmp.insert(x);
size_t len = tmp.size();
nums.erase(nums.begin(),nums.end());
for(auto x:tmp)
nums.push_back(x);
return len;
}
};

title:leetcode：Same Tree
date:2016-05-30 04:41:44
/**

 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if(!p && !q)
            return true;
        if(!p || !q)
            return false;
        if(p->val != q->val)
            return false;

        bool l(isSameTree(p->left,q->left));
        bool r(isSameTree(q->right,p->right));
        return l&&r;
    }
};

title:locale.Error: unsupported locale setting
date:2016-10-05 03:29:40
今天在服务器上装shadowsocks时，出现了如下的错误：

Traceback (most recent call last):
File “/usr/bin/pip”, line 11, in
sys.exit(main())
File “/usr/lib/python2.7/dist-packages/pip/init.py”, line 215, in main
locale.setlocale(locale.LC_ALL, ‘’)
File “/usr/lib/python2.7/locale.py”, line 581, in setlocale
return _setlocale(category, locale)
locale.Error: unsupported setting

从字面上看，是LC_ALL环境变量无法确定导致的，在终端输入：

LC_ALL=C

清除所有本地化设置，程序成功运行。

在Linux中通过locale来设置程序运行的不同语言环境，locale由ANSI C提供支持。locale的命名规则为<语言>_<地区>.<字符集编码>，如zh_CN.UTF-8，zh代表中文，CN代表大陆地区，UTF-8表示字符集。在locale环境中，有一组变量，代表国际化环境中的不同设置，而LC_ALL代表一个宏，用于设置所有的LC变量。
