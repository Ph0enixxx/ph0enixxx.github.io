bool finds(int p[], int len ,int target)

title:二次快速幂(c++\Python)
date:2016-01-23 03:36:52
快速幂的思想是将指数分解，然后将几个小的指数得到的结果合并 来达到加速开方的目的。

百度百科上的二次快速幂：

#include <iostream>
int pow2(int ,int );
int main(void)
{
    std::cout<<pow2(2,7);
}
int pow2(int num,int s)
{
    int r = 1,base = num;
    while(s != 0)
    {
        if(s%2)//如果是奇数，就先把结果乘以自身（整理为一堆偶数的快速幂）
            r*=base;
        base*=base;//base变为base的平方
        s/=2;
    }
    return r;
}

计算步骤：
2的7次幂：
1.7是奇数 r = 2
2.base = 2^2 = 4
3.s = 3
4.3是奇数 r = 8
5.base = 4^4 = 16
6.s = 1
7.1是奇数 r = 128
8.s = 0 返回r结束
计算步骤只有 2， 5 ，7 三步
相比普通的7步快了接近一倍

Python:

def pow(num , s):
    r = 1
    base = num
    while s != 0 :
        if s%2 == 1:
            r *= base
        base *= base
        s /= 2
        s = int(s)
    return r
print(pow(2,9))

title:二分图的判定

#include <stdio.h>
#include <iostream>
#include <map>
#include <vector>
#include <algorithm>
using std::vector;
using std::map;

std::map<int, std::vector<int> > data;
bool *color;
bool is2f(true);

void init()
{
  // for(int i=0;i<100;i++)
  // {
  //   for(int k=0;k<=100;k++)
  //     {
  //       data[i].push_back(k);
  //     }
  // }
  data[0].push_back(1);
  data[1].push_back(2);
  data[1].push_back(3);
  //data[2].push_back(3);
}


bool *visited;
void dfsColor(int n,bool clr=true)
{

  if(NULL == color)
    color = new bool[data.size()];
  if(NULL == visited)
    visited = new bool[data.size()];
  if(visited[n] == true)
    return;
  else
    visited[n] = true;
  std::cout<<"->"<<n;
  for (int i = 0; i < data[n].size(); ++i)
  {
    if(color[data[n][i]]==clr)
    {
      is2f = false;
      return;
    }
    dfsColor(data[n][i],1-clr);
  }
}

int main(int argc, char const *argv[])
{
  /* code */
  init();
  //dfs(12);
  dfsColor(1);
  std::cout<<std::endl<<is2f;
  return 0;
}

原理：在深度搜索的同时，为每一个节点标记与其父节点不同的颜色，如果遇到相同颜色的节点相邻，则判断为不是二分图，否则是二分图。

title:优盘安装ubuntu server
date:2016-04-10 01:43:29
用U盘安装ubuntu server时，会出现找不到光盘的情况：

解决方案：

1.切换到shell运行环境

2.cd /dev/

3.ls sd 此时看看最后两个的sd是什么 比如（sdd sdd4）

4.cd / mount /dev/sdd4 /usb（刚刚出现的那个）

5.将你优盘中的iso文件挂载到cdrom即可：

mount /usb/ubuntu-server-xxxxx.iso /cdrom

exit
