title:collections学习笔记
date:2017-02-06 05:28:18

namedtuple：

为tuple命名<—> 类似于C语言的结构体

根据条件生成一个类

官方的解释是nametuple是一个根据命名字段来产生tuple子类的工厂函数

Point = namedtuple('Point',['x','y','z'])

p = Point(1,2,3)

p.x

    1

p.y

    2

deque:

同stl中的deque，为了快速从两端插入删除元素的双向链表

其中的一个很方便的功能是extend方法，可以将其他可迭代的数据结构拼接上去

defaultDict：

这个数据结构主要是处理不存在的元素的问题，当访问一个不存在的key时，会给出初始化的默认值或者执行一段写好的函数

内部实现代码还没有看 ，我猜想是重载python字典的__getitems__和__getattr__魔术方法

OrderedDict：

有序字典解决了Python中最大的痛处之一：dict没有顺序，注意一点是两个有序字典对象只有在顺序和键值对完全相等的情况下，才能相等

Counter：

Counter是一个dict的子类，作为计数器用时，省去了将dict的key不存在时将其置为1的操作（个人认为还是用c++中的map<std::string,int>或者hashmap实现比较好）

title:hdu 蟠桃记
date:2016-01-21 10:46:08

Problem Description
喜欢西游记的同学肯定都知道悟空偷吃蟠桃的故事，你们一定都觉得这猴子太闹腾了，其实你们是有所不知：悟空是在研究一个数学问题！
什么问题？他研究的问题是蟠桃一共有多少个！
不过，到最后，他还是没能解决这个难题，呵呵^-^
当时的情况是这样的：
第一天悟空吃掉桃子总数一半多一个，第二天又将剩下的桃子吃掉一半多一个，以后每天吃掉前一天剩下的一半多一个，到第n天准备吃的时候只剩下一个桃子。聪明的你，请帮悟空算一下，他第一天开始吃的时候桃子一共有多少个呢？
Input
输入数据有多组，每组占一行，包含一个正整数n（1<n<30），表示只剩下一个桃子的时候是在第n天发生的。
Output
对于每组输入数据，输出第一天开始吃的时候桃子的总数，每个测试实例占一行。

Sample Input
2
4
Sample Output
4
22

title:leetcode: Contains Duplicate
date:2016-06-08 08:46:53
class Solution {
public:
bool containsDuplicate(vector& nums) {
map tmp;
for(auto x:nums)
{
if(tmp[x] == false)
tmp[x] = true;
else
return true;
}
return false;
}
};

title:leetcode: Number of 1 Bits
date:2016-05-29 04:27:36
class Solution {
public:
int hammingWeight(uint32_t n) {
int cnt(0);
while(n!=0)
{
cnt += n&1;
n >>= 1;
}
return cnt;
}
};

要求是计算一个数转换成2进制后的1的个数，利用异或的特点加上移位运算就能解出。
