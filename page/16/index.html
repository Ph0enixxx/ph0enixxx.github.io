设置iptables

iptables -I INPUT -p tcp --dport 80 -m connlimit --connlimit-above 30 -j REJECT

title:搭建Git服务器教程
date:2016-03-04 05:26:06
1.安装git（以Ubuntu示例）

apt-get install git

2.切换到管理员账户，执行添加用户命令：

adduser git

3.改密码

passwd git

4.制作用户目录：

mkdir /home/git

chown git git

chgrp git git

5.切换到git用户并进入/home/git目录

6.创建代码目录 并在该目录下执行 git -init

7.现在可以通过git clone git@git:202.194.119.110:~/code来访问了
